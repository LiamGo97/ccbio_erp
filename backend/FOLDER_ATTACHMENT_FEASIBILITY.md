# 폴더 첨부 기능 가능성 검토

## 요구사항
- 제품 사진들이 Google Drive 폴더에 모여있음
- 폴더를 첨부하여 스케줄에 연결하고 싶음

## 기술적 가능성 분석

### 1. Google Drive API 기능
✅ **가능**: Google Drive API는 폴더 내의 모든 파일을 재귀적으로 조회할 수 있습니다.

#### 필요한 API 메서드:
- `listFiles` (현재 존재): 특정 폴더의 직접 자식만 조회
- `listFilesRecursively` (새로 추가 필요): 폴더 내의 모든 파일을 재귀적으로 조회
  - 하위 폴더까지 모두 탐색
  - 모든 파일의 ID, 이름, 경로 정보 수집

### 2. 데이터베이스 구조
✅ **가능**: 현재 구조를 확장하여 폴더 ID를 저장할 수 있습니다.

#### 현재 구조:
- `tb_trade_contract`: `tc_contract_google_drive_file_id` (단일 파일 ID)
- `tb_trade_order`: `to_invoice_google_drive_file_id` (단일 파일 ID)

#### 확장 방안:
**옵션 1: 기존 컬럼 사용 (단순)**
- `contractGoogleDriveFileId`에 폴더 ID도 저장 가능
- 파일/폴더 구분을 위해 타입 컬럼 추가 필요:
  - `tc_contract_attachment_type` (ENUM: 'FILE' | 'FOLDER')

**옵션 2: 별도 컬럼 추가 (명확)**
- `tc_contract_google_drive_folder_id` (폴더 ID)
- 기존 `tc_contract_google_drive_file_id`와 구분

**옵션 3: 제품 사진용 별도 테이블 (확장성)**
- `tb_trade_order_product_images` 테이블 생성
- `to_id` (스케줄 ID), `google_drive_folder_id`, `folder_name` 등

### 3. 프론트엔드 UI
✅ **가능**: 현재 `GoogleDriveFilePicker`를 확장하여 폴더 선택 가능

#### 필요한 변경사항:
1. **파일 선택기 수정**:
   - 폴더 선택 모드 추가 (파일/폴더 선택 옵션)
   - 폴더 선택 시 폴더 아이콘 및 폴더 정보 표시

2. **스케줄 폼 수정**:
   - "제품 사진 폴더" 필드 추가
   - 폴더 선택 버튼 및 선택된 폴더 표시
   - 폴더 미리보기 (폴더 내 파일 목록)

### 4. 백엔드 API
✅ **가능**: 폴더 내 모든 파일을 조회하는 API 추가

#### 필요한 API 엔드포인트:
1. **폴더 내 모든 파일 조회**:
   ```
   GET /google-drive/folders/:folderId/files?recursive=true
   ```
   - 폴더 ID를 받아서 내부의 모든 파일을 재귀적으로 조회
   - 파일 목록과 경로 정보 반환

2. **스케줄 저장 시 폴더 ID 저장**:
   - 기존 `contractGoogleDriveFileId` 대신 또는 추가로
   - `contractGoogleDriveFolderId` 필드 추가
