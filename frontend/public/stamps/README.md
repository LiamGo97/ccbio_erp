# 도장 파일 저장 위치

이 폴더에는 거래명세서에 사용되는 도장 이미지 파일을 저장합니다.

## 파일명
- `stamp.png` 또는 `stamp.jpg`: 거래명세서 하단 "인수" 부분에 표시되는 도장 이미지

## 사용 방법
1. 도장 이미지 파일을 이 폴더에 저장하세요.
2. 파일명은 `stamp.png` 또는 `stamp.jpg`로 저장하세요.
3. 거래명세서에서 자동으로 표시됩니다.

## 파일 형식
- PNG 또는 JPG 형식 권장
- 권장 크기: 80x80px ~ 100x100px
- 투명 배경 PNG 권장
